# NBA-Win-Prediction-Analysis
Analyzed NBA team performance using R to identify key basketball statistics/metrics that best predict winning. Performed data cleaning, exploratory analysis, and regression modeling, and ended up finding offensive and defensive ratings as the strongest predictors. Applied many data science and statistical modeling skills like linear regression.

## Overview
This project analyzes which team efficiency metrics best predict winning in the NBA using statistical modeling and data analysis in R.

## Dataset
- Source: Kaggle NBA Team Statistics Dataset
- Contains team-level data across multiple NBA seasons

## Key Variables
- Offensive Rating (o_rtg)
- Defensive Rating (d_rtg)
- Net Rating (n_rtg)
- Margin of Victory (mov)
- Turnover Percentage (tov_percent)

## Methods
- Data Cleaning (handling missing values, filtering post-2000 seasons)
- Exploratory Data Analysis (histograms, scatter plots)
- Correlation Analysis
- Linear Regression Modeling

## Key Findings
- Offensive and Defensive Rating are the strongest predictors of wins
- Net Rating shows extremely high correlation with success
- Turnover percentage has weaker influence

## Tools Used
- R
- R Markdown
- ggplot2 / base R plotting
